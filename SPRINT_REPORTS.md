## 📝 使用说明

### Web 界面配置股票

1. 访问 `http://localhost:5000`
2. 滚动到页面底部"⚙️ 股票配置管理"
3. 填写表单添加股票：
   - 股票代码（如：600519）
   - 股票名称（如：贵州茅台）
   - 市场（上海/深圳）
   - 别名（可选）
   - 备注（可选）
4. 点击"➕ 添加股票"

### Cron 管理

```bash
# 安装预警任务
./install_alert_cron.sh install

# 查看状态
./install_alert_cron.sh status

# 测试执行
./install_alert_cron.sh test

# 卸载任务
./install_alert_cron.sh uninstall
```
